Les sources java de la librairie sont toutes dans le répertoire src/.

- "sh scripts/run.sh" pour lancer le programme avec un nombre d'éléments donné
- "sh scripts/makejar.sh" pour créer un jar contenant tous les .class
- "sh scripts/clean.sh" pour supprimer tous les fichiers générés (sauf les librairies)



Une fois en jeux les boutons (rouge,bleu,vert,jaune) servirons a selectionner quel pion vous souhaitez deplacer.
Les boutons (gauche,bas,haut,droite) permetterons de se déplacer, le bouton actualisé permettra de generer une nouvelle grille, puis enfin le bouton résoudre pour trouver la solution qui sera afficher dans le terminal.

En ce qui concerne notre solveur il fonctionne uniquement en déplaçant 1 robot, quand il ne fonctionnera pas un message d'erreur apparaîtra.
